# Strategy Cards for Claude Code Workshop

## Overview

Strategy cards are quick-reference guides that participants can use during challenges. They can be printed as physical cards or provided digitally.

## Card Dimensions
- Recommended size: 4" x 6" (index card size)
- Can be printed double-sided
- Laminate for durability

## Usage
- Distribute at the start of the workshop
- Reference during challenges
- Keep for future use

## Digital Format

Each card is provided as a markdown file that can be:
- Printed as PDF
- Displayed on screen
- Shared digitally

## Printing Instructions

1. Print on cardstock (110 lb recommended)
2. Cut along guides
3. Optional: Laminate for durability
4. Hole punch corner for ring binding (optional)
